# KOVA System v2

KOVA System v2 is the rebuilt platform for the KOVA Valorant organization.

## Workspace

This repository is organized as a monorepo with separate applications and shared packages.

```text
apps/
  admin-web/
  apply-web/
  backend/
  discord-bot/
  main-web/
packages/
  config/
  database/
  shared/
  ui/
docs/
```

## Current Status

The v1 bot and v1 apply site are kept in this workspace as reference-only material.

The active v2 planning document is:

- `KOVA_V2_SPEC.md`

OAuth setup for the apply website:

- `docs/DISCORD_OAUTH_SETUP.md`

## Build Order

1. Backend
2. Discord bot
3. Apply website
4. Admin panel
5. Main website
